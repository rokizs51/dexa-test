# DEXA SWE TEST DESIGN

1. Requirements
    1. Frontend : React.js
    2. Backend : NestJs (Typescript), Mysql, DrizzleORM
    
    Architect : Microservice
    
    Objectives : 
    
    - Able Design DB, Connect DB (ORM), Manipulate Data ( CRUD )
    - Create Page using JS framework, create custom component, call api from / to backend
2. Cases  :
    1. Aplikasi Absensi WFH Karyawan
    Buat sebuah aplikasi web, dimana memungkinkan karyawan dapat melakukan login,
    dan juga absen menggunakan aplikasi tsb, dengan meng- capture tanggal & waktu, dan juga
    melakukan upload foto sebagai bukti karyawan sedang bekerja di rumah.
    2. Aplikasi Monitoring Karyawan
    Untuk data master karyawan dapat dibuat menggunakan aplikasi berbasis web,
    dimana nantinya admin HRD dapat melakukan penambahan ataupun melakukan update
    terhadap data karyawan, dan juga melakukan kontrol (view only) terhadap absensi yang
    telah disubmit oleh karyawan.
